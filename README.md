# YReason (Formal Explainer for Rule Ensembles)

The source code for paper "Efficient Explanations for Rule Ensembles" accepted in CP 2026.
The code structure majorly follows the [XReason](https://github.com/alexeyignatiev/xreason). 

## Getting Started

We recommend the python virtual environment to install all packages. 
The python version recommended is 3.13.7.
All necessary packages are included in requirements.txt.

```bash
# create the virtual environment naming .venv
python3 -m venv .venv

# activate the virtual environment (under macos/linux)
source .venv/bin/activate

# install all packages under the venv
pip install -r requirements.txt
```
